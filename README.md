# Development

## Install or update local packages

```bash
npm run install
```

## Run the project

```bash
npm run dev
```

## Check code quality and style

To ensure code quality and seamless development, eslint and prettier are integrated in the codebase.

### Code quality (linter)

```bash
npm run eslint
```

### Code style (prettier)

```bash
npm run format
```
